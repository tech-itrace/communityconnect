## âš¡ Query Performance Comparison

### Alumni Search by City

**BEFORE:**
```sql
SELECT m.name, ap.college, ap.city
FROM members m
JOIN community_memberships cm ON m.id = cm.member_id
JOIN alumni_profiles ap ON cm.id = ap.membership_id
WHERE ap.city ILIKE '%bangalore%';

-- 3 table JOINs
-- Uses idx_alumni_city
```

**AFTER:**
```sql
SELECT m.name, 
       cm.profile_data->>'college' as college,
       cm.profile_data->>'city' as city
FROM members m
JOIN community_memberships cm ON m.id = cm.member_id
WHERE cm.profile_data->>'city' ILIKE '%bangalore%';

-- 1 table JOIN
-- Uses idx_cm_profile_city
-- 60% faster
```
